# Application dependencies

```
- Node v16.15.1 (LTS) or more recent. While older versions can work it is advisable to keep node to latest LTS version

- npm 8.11.0 (LTS) or more recent, Yarn can work but was not tested for this project

- AWS CLI - aws-cli/1.16.309 Python/3.6.0 Windows/10 botocore/1.13.45

- EB CLI - 3.20.3

- (Optional) psql - version 12.11

- A RDS database running Postgres.

- A S3 bucket for hosting of Storefront Frontend.

- AWS ElasticBeanstalk forStorefront Backend deployment

```
